A financial institution needs to host a secure internal application that handles sensitive client data. The project utilizes EC2 instances within a VPC to create isolated environments for hosting the application.

Project Title
Harnessing EC2, VPC & MobaXterm for Cutting-Edge Entity Hosting

👉 Live Website URL: http://51.20.51.104/landing-page/#home

Project Description
This project demonstrates hosting a static website on AWS EC2 instance using VPC networking and secure remote access through MobaXterm.

The website is deployed on an EC2 instance and is accessible publicly using a static IP address.

Features
Hosted on AWS EC2 instance
Secure access using MobaXterm SSH
Configured inside VPC network
Static website deployment (HTML, CSS, JS)
Public IP based access
Tech Stack
AWS EC2
AWS VPC
Linux Server (Amazon Linux)
MobaXterm (SSH Client)
HTML, CSS, JavaScript
